# Hangman Game

A modern, interactive Hangman game built with Python Flask, featuring a clean web interface with keyboard support and visual hangman drawings

## âœ¨ Features

- **Web-based interface** - Play directly in your browser
- **Visual Feedback** - ASCII art hangman that updates with each wrong guess
- **Data input methods** - On-screen keyboard + physical keyboard support
- **Responsive design** - Works on desktop, tablet, and mobile
- **Session management** - Game state persists during your session
- **Win/Lose detection** - Automatic game outcome determination
- **Easy Customization** - Modify word lists and game settings
